## Overview

An interactive, bilingual (EN/TR) scrollytelling documentary about untold stories from World War II. Every element — photographs, maps, narration, text, soundtrack — was generated by AI through conversational prompts. No hand-written code, no manual asset creation.

**Live URL:** https://albatrosjj.github.io/hermes-living-doc/

## Key Features

### 🎭 Cinematic Start Screen
"Enter Experience" button unlocks audio context (required by browser autoplay policies) and initiates the immersive journey.

### 🗣️ Scroll-Triggered Audio
Each section auto-plays its narration when it enters the viewport. Other sections pause automatically. Background music ducks (80% volume) during narration.

### 🌐 Instant Language Switching
One-click EN/TR toggle. All text, titles, dates, captions, and audio sources swap without page reload.

### 🌎 Combatant Flags & Maps
Every chapter shows which nations fought, with emoji flags and colored borders (Allied / Axis / Resistance). Plus a vintage-style AI-generated military map.

### 🎨 Film Grain Overlay
Subtle SVG noise texture fixed over the entire viewport for authentic documentary feel.

### 📊 Progress Bar & Chapter Navigation
Top progress bar tracks scroll position. Right-side dot navigation jumps between chapters.

## How It Was Built

All commands executed through Hermes Agent conversational interface. No IDE, no manual coding.

1. **Research:** WWII untold stories identified via web search
2. **Outline:** JSON structure with EN/TR text, prompts, flags, epilogues
3. **Photos:** 8 AI images generated via FAL.ai (`image_generate` tool)
4. **Maps:** 8 vintage-style maps generated via FAL.ai
5. **Audio:** 16 narration files via `text_to_speech` (Edge TTS)
6. **Code:** HTML/CSS/JS generated and patched via `write_file`/`patch` tools
7. **Build:** `python3 generate.py` assembles single-page app
8. **Deploy:** Git push to GitHub Pages
