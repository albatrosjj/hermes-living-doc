# The Silence Between Thunder: A Living Documentary

**Hermes Agent + Kimi Moonshot AI Hackathon Project**

Fully AI-generated bilingual (EN/TR) scrollytelling documentary about WWII untold stories.

## Features
- TR/EN instant language switch
- Auto-play narration on scroll
- Vintage military maps per chapter
- Combatant flags & front details
- Cinematic film grain & parallax
- Artistic poetic epilogues

## Tech Stack
- Hermes Agent (Kimi K2.6)
- FLUX (FAL.ai) for images & maps
- Edge TTS for narration
